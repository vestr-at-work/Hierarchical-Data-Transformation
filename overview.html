Hierarchical data transformation Overview

This is the documentation for the Hierarchical data transformation project. This project provides a tool for transforming hierarchical data converted to the Unified Representations (Ur). Includes operations such as filtering, default value assignments, subtrees shifting and more.

Modules

The project is divided into several modules:

  • cz.cuni.mff.hdt.transformation: Contains core transformation functionality.
  • cz.cuni.mff.hdt.operation: Contains operations that can be performed on the Ur data.
  • cz.cuni.mff.hdt.converter: Provides converters to and from the Unified Representation.
  • cz.cuni.mff.hdt.ur: Defines the core data structures and utilities for Unified Representation.
